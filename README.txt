Click to chat, contact, message

== Description ==
There are 18 contact channels: WhatsApp, Email, Instagram, Line, custom link, LinkedIn, Map, Facebook Message, phone, Skype, SMS, Snapchat, Telegram, Twitter, Viber, Vkontakte, Waze, Wechat

Beautifully designed, there are many options for icon styles, colors, positions, and display styles that make it easy to adapt to any web page.

== Installation ==
Select the contact channel and fill in the information, then select the color, display location, display type, icon type and save. The contact channels filled in will be displayed outside the frontend page.
